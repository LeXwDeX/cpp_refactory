# 工具缺口登记

> 重构过程中发现的工具不足，在此登记和追踪。
> 发现即登记，修复即更新，不允许事后补录。

## 缺口类型

| 类型 | 说明 |
|---|---|
| A | 工具缺失（command not found） |
| B | 工具产出不符合预期 |
| C | 手动重复操作（>3 次未脚本化） |
| D | 脚本报错 |
| E | 精度不足（误报率 >30%） |

## 总览

- **总缺口数**：0
- **OPEN**：0
- **IN_PROGRESS**：0
- **CLOSED**：0
- **ESCALATED**：0
